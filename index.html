Proyecto

El proyecto trata de un control de asistencia automatizado, el cual funciona junto con una pagina web. El proyecto se basa en la creacion de un sistema de control de asistencia atumatizado, controlado por tarjetas Rfid y Arduino, con una base de datos cargada en una pagina web, donde se podra ingresar a ver los registros cuando se lo desee.

El objetivo principal es agilizar el proceso del control de presencialidad ayudando así a los profesores a tener un registro de asistencia cargado en una aplicación web y así ayudar a saber que alumnos estuvieron presentes ese día.
Como objetivo secundario, Tener control de que alumnos pasaron por tal salon en tal momento, tambien poder conocer la entrada del personal del colegio diferenciando la entrada a la escuela y a los salones.
